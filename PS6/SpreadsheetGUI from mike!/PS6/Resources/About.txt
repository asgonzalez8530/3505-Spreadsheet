About:
	Purpose:
		The purpose of the is program is to track words, numbers, and formulas. When specified with an "=", the 
		program will attempt to calculate the given formula. This program is a simplified version of excel. All projects
		made in this program are saved with a .sprd tag. You can open mulitple windows of this program. Other features 
		include saving, opening, and closing a spreadsheet file.
